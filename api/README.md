# API Service (FastAPI)

## Quick Start

```bash
cd api
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

Open: http://127.0.0.1:8000/docs
